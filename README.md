Teen Depression Analysis Dashboard
Project Overview

This project analyzes student mental health data using Power BI. The dashboard provides insights into depression scores, anxiety levels, sleep patterns, study habits, attendance, stress levels, and burnout among students.

Tools Used
Power BI
Data Visualization
Data Analysis
Key Features
Interactive dashboard with KPI cards
Analysis of depression and anxiety trends
Sleep pattern and study hour insights
Stress level distribution visualization
Attendance and burnout analysis
Dataset Information

The dataset contains student-related mental health indicators including depression score, anxiety score, sleep hours, study hours, attendance percentage, stress level, burnout level, and CGPA.

Key Insights
Average anxiety score: 5.49
Average daily sleep hours: 6.50
Comparison of stress levels among students
Relationship between sleep patterns and depression indicators
Impact of attendance on burnout levels
